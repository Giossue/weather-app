# Meteocons

Estos íconos climáticos fueron extraídos de `@meteocons/svg` / https://github.com/basmilius/meteocons.

- Autor: Bas Milius
- Licencia: MIT
- Estilo usado: `fill`
- Formato: SVG animado con SMIL (`<animate>` / `<animateTransform>`)

La licencia MIT está copiada en `METEOCONS-LICENSE`.

Mapeo usado por la app:

| OpenWeather | Meteocons animado |
| --- | --- |
| `01d` | `clear-day.svg` |
| `01n` | `clear-night.svg` |
| `02d` | `mostly-clear-day.svg` |
| `02n` | `mostly-clear-night.svg` |
| `03d` | `partly-cloudy-day.svg` |
| `03n` | `partly-cloudy-night.svg` |
| `04d` | `overcast-day.svg` |
| `04n` | `overcast-night.svg` |
| `09d`, `09n` | `drizzle.svg` |
| `10d`, `10n` | `rain.svg` |
| `11d`, `11n` | `thunderstorms-rain.svg` |
| `13d`, `13n` | `snow.svg` |
| `50d`, `50n` | `fog.svg` |

Nota: para lluvia, tormenta, nieve y niebla se usan íconos genéricos sin sol/luna para evitar contradicciones visuales como “lluvia ligera” con sol.
