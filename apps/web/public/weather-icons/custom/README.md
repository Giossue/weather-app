# Íconos personalizados

Reemplaza estos archivos por tus propios íconos manteniendo el código oficial de OpenWeather.

- Día: `day/01d.svg`, `day/10d.svg`, etc.
- Noche: `night/01n.svg`, `night/10n.svg`, etc.
- También puedes usar `.png` o `.webp` configurando la ruta en `weather-icon-map.ts`.

Si un archivo personalizado falla al cargar, el componente usa `public/weather-icons/fallback`.
